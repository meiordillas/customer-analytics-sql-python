# Customer Analytics Project (SQL + Python)

## Overview
This project demonstrates end-to-end data analysis using SQL and Python. The goal is to analyze customer behavior, conversion funnels, and revenue performance using an e-commerce dataset.

## Objectives
- Analyze customer purchase behavior
- Identify top-performing segments
- Calculate conversion rates and revenue metrics
- Demonstrate SQL proficiency and Python-based analysis

## Tools Used
- SQL (BigQuery / Snowflake compatible)
- Python (Pandas, Matplotlib)
- Jupyter Notebook

## Dataset
Synthetic e-commerce dataset including:
- Customer ID
- Order ID
- Order Date
- Revenue
- Traffic Source
- Device Type

---

## Key Analyses

### 1. Revenue by Traffic Source
- Identifies which channels drive the most revenue

### 2. Conversion Funnel
- Tracks user progression from visit to purchase

### 3. Customer Segmentation
- Groups users by behavior and value

### 4. Cohort Analysis
- Measures retention over time

---

## Sample Insights
- Organic traffic generated the highest total revenue
- Mobile users had higher traffic but lower conversion rates
- Returning customers contributed significantly to total revenue

---

## How to Run

### SQL
Run queries in `sql/analysis.sql` using your preferred SQL environment.

### Python
Open `python/analysis.ipynb` in Jupyter Notebook and run all cells.
